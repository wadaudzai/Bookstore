# Math Magicians

> This project was built to display a simple react app with components there are several parts of this project which display books, a form to submit it and other functionalities but it isn't complete yet so unfortunately I cannot give you more information.


## Built With

- Javascript
- React
- Webpack, Babel


## Getting Started

To start with this project. You have to run npm install. When you want to build the project run npm build and to start the project and run it on a browser run npm start. 

### Prerequisites

You just need a good computer and a code editor.
